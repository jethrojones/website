Yesterday I worked with Codex on three small projects. I updated a service website with clearer payment, refund, cancellation, privacy, and support information. I fixed a support request flow so new requests are assigned to the right person. I also updated an admin export so it pulls the needed records more directly.<!--more-->

The website work included a new policies page, updates to the terms and privacy pages, clearer price labels, and links from the public pages to the policy information.

The support flow work made sure new help requests create an assigned task instead of sitting only as a dashboard message.

The export work changed the report source so the download includes the right set of records. I also ran targeted tests on the support and export changes.
